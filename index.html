About this project

       ในปัจจุบันนี้ แม้เทคโนโลยีจะพัฒนาไปมากแค่ไหนแต่เรื่อง ความเชื่อ ที่เป็นสิ่งยึดเหนี่ยวจิตใจของคนเรา ก็ยังมีอิทธิพลเป็นอย่างมาก เช่น การใส่เสี้อตามสีตามวัน สวมเครื่องรางมงคล และการดูดวง ก็คือหนึ่งในกลุ่มของพวกเราจึงเลือกให้ความสนใจเก็บการดูดวงจาก ไพ่ทาโร่เกิดเป็นโปรแกรม Horoscope Program โปรแกรมนี้ใช้ภาษา C ในการเขียน โดยให้ผู้ใช้ทำการกรอกชื่อและเลือกด้านที่อยากรู้พร้อมทั้งเลือกไพ่ 3 ใบ โปรแกรมจะทำการสุ่มคำทำนายจากเลขไพ่ที่ผู้ใช้ได้เลือก และแสดงผลคำทำนาย จากการทดลองใช้ทำให้ทราบว่าโปรแกรมนี้สามารถทำนายดวงได้จริง ทั้งเรื่องความรัก การเงิน การงาน และสุขภาพได้อย่างแม่นยำ






โปรแกรมนี้ทำอะไร

ดูดวงด้านต่างๆ ทั้งการเงิน การงาน ความรักและสุขภาพ ให้ผู้ใช้ได้สัมผัสประสบการณ์การดูดวงที่เหมือนจริงโดยการเลือกไพ่








❝ หากคุณกำลังมีเรื่องที่สงสัยไม่ว่าจะเป็นในอดีต ปัจจุบัน และอนาคตข้างหน้า

โปรแกรมนี้คือสิ่งที่ตอบโจทย์สำหรับคุณ ❞

ดูดวงแทบตาย สุดท้ายก็ไปกับเขา

How to use

1. เมื่อเปิดโปรแกรมขึ้นมา ให้ทำการกรอกชื่อของคุณ
2. เลือกหมวดที่คุณต้องการดูดวง เช่น ความรัก การเงิน สุขภาพ การทำงาน หรือ การอ่านไพ่เพียงอย่างเดียว
3. เมื่อทำการเลือกหมวดหมู่จะให้ท่านทำการเลขตัวเลข 3 ตัวที่ไม่ซ้ำกันตั้งแต่ 0 ถึง 21 เปรียบเสมือนการเลือกไพ่ขึ้นมา
4. กด Enter เพื่อแสดงคำทำนายที่ได้
5. เมื่ออ่านเสร็จ หากต้องการดูดวงในเรื่องอื่นต่อ ให้พิมพ์ “Y” หากไม่ต้องการให้พิมพ์ “N”

ตัวอย่างการทำงานของโปรแกรม
